**Java Basics**



A collection of simple Java programs for learning core concepts like input/output, conditions, loops, and basic algorithms.




*Programs Included*

\- EvenOdd.java

\- Greeting.java

\- SimpleInterest.java

\- Calculator.java

\- Largest.java

\- RupeetoUSD.java

\- Fibonacci.java

\- Palindrome.java

\- Armstrong.java




*How to run*

1\. Install Java (JDK 8+)

2\. Open terminal in this folder

3\. Compile a file:

&nbsp;  javac FileName.java

4\. Run:

&nbsp;  java FileName
